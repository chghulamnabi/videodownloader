Getting Started

Welcome to the YTD Video Download API. This API allows you to download videos from YouTube, TikTok, and Vimeo programmatically. All platforms have been tested and verified working.

Verified Platforms: YouTube (videos & shorts), TikTok, Vimeo
Not Supported: Instagram, Facebook, Twitter/X, Dailymotion (require authentication)

Base URL: https://api.ytd.com (Change to your domain)

All API requests must be made over HTTPS. Calls made over HTTP will fail.

Authentication

Authenticate your API requests by including your API key in the request headers or as a query parameter.

Header Authentication (Recommended)

X-API-Key: your_api_key_here

Query Parameter Authentication

?api_key=your_api_key_here
Demo API Key: demo_key_123
(50 requests per day, for testing only)

Available Endpoints

POST /api/video/info

Get comprehensive information about a video including title, author, duration, and available quality formats.

Request Body

Parameter Type Required Description
url string Yes Video URL from supported platform

Example Request

curl -X POST https://api.ytd.com/api/video/info \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key_here" \
  -d '{
    "url": "https://youtube.com/watch?v=dQw4w9WgXcQ"
  }'

Example Response

POST /api/video/download

Download a video in specified quality. Returns the video file as a stream.

Request Body

Parameter Type Required Description
url string Yes Video URL from supported platform
quality string No Desired quality (360p, 720p, 1080p). Default: 720p
format string No Output format (mp4, webm). Default: mp4

Example Request

curl -X POST https://api.ytd.com/api/video/download \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key_here" \
  -d '{
    "url": "https://youtube.com/watch?v=dQw4w9WgXcQ",
    "quality": "1080p",
    "format": "mp4"
  }' \
  --output video.mp4

Response Headers

Content-Type: video/mp4
Content-Disposition: attachment; filename="video_title.mp4"
X-Remaining-Requests: 49

GET /api/platforms

Get list of all supported platforms and their capabilities.

Example Request

curl -X GET https://api.ytd.com/api/platforms \
  -H "X-API-Key: your_api_key_here"

Error Handling

The API uses standard HTTP status codes to indicate success or failure.

Status Code Meaning
200 Success
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error

Error Response Example

{
  "success": false,
  "error": "Rate limit exceeded",
  "message": "You have exceeded your daily request limit. Upgrade your plan."
}

Code Examples

JavaScript

const response = await fetch('https://api.ytd.com/api/video/info', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key_here'
  },
  body: JSON.stringify({
    url: 'https://youtube.com/watch?v=dQw4w9WgXcQ'
  })
});

const data = await response.json();
console.log(data.data.title);

Python

import requests

response = requests.post(
    'https://api.ytd.com/api/video/info',
    headers={'X-API-Key': 'your_api_key_here'},
    json={'url': 'https://youtube.com/watch?v=dQw4w9WgXcQ'}
)

data = response.json()
print(data['data']['title'])

PHP

$ch = curl_init('https://api.ytd.com/api/video/info');
curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'url' => 'https://youtube.com/watch?v=dQw4w9WgXcQ'
]));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'X-API-Key: your_api_key_here'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);
echo $data['data']['title'];
