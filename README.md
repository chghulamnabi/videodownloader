# 🎥 YouTube Video Downloader - Advanced UI

A modern, feature-rich YouTube video downloader with a stunning glassmorphism UI design.

## ✨ Features

- **Modern UI/UX Design**: Glassmorphism effects, animated gradients, smooth transitions
- **Multiple Formats**: Download videos in various qualities (144p to 4K)
- **Audio Extraction**: Download audio-only files
- **Video Preview**: See video information before downloading
- **Real-time Progress**: Visual download progress indicator
- **Responsive Design**: Works perfectly on all devices
- **Fast & Secure**: Direct downloads, no data collection

## 🚀 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Setup Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start the Server**
   ```bash
   npm start
   ```

3. **Open Your Browser**
   Navigate to: `http://localhost:3000`

## 📦 Dependencies

- **express**: Web server framework
- **cors**: Cross-origin resource sharing
- **ytdl-core**: YouTube video downloader library

## 🎨 UI Features

- Animated gradient background with floating orbs
- Glassmorphism card effects
- Smooth animations and transitions
- Interactive hover effects
- Progress indicators with shimmer animation
- Responsive design for all screen sizes

## 🔧 Usage

1. **Enter YouTube URL**: Paste any YouTube video URL in the input field
2. **Get Video Info**: Click "Get Video" to fetch video details
3. **Select Format**: Choose between Video or Audio-only
4. **Select Quality**: Pick your preferred quality/resolution
5. **Download**: Click on a quality option to start downloading

## 📝 Supported URLs

- Standard YouTube videos: `https://www.youtube.com/watch?v=...`
- Short URLs: `https://youtu.be/...`
- YouTube Shorts: `https://www.youtube.com/shorts/...`
